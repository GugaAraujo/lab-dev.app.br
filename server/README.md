
## Description

The project server uses the [Nest](https://github.com/nestjs/nest) framework.
We recommend using the [documentation](https://docs.nestjs.com/) as a reference.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
